# Spotify App 
This is a Cloned version of Spotify App using Swift. 
This is only for personal education purpose. This clone is featuring most of the features that Spotify has.



# Technologies & Features 
- AutoLayout
- UIStackView
- Adaptive UI
- Progarammatic UI
- No Storyboard approach
- UISwitch
- UILabels
- UIButtons
